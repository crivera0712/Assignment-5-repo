How to Apply CSS to HTML

To be able to apply external CSS to an html file you have to use an external style sheet. External stylesheets have a ".css" extension. A single CSS file can be linked to multiple html files. To link external CSS file to an html file you need to create a <link> element. For example: <link rel = "stylesheet" href="style.css">

How to Use JavaScript in a Webpage

There are multiple ways to use JavaScript code in an html Webpage. One way to implement JS in an html files is by linking an external script. The <script> element allows you to execute an external .js script by linking it. For example: <script src="path/to/script.js"><script>. Another way to use JS in an html file is by writing JS within the .html file. This can also be done with the <script> element. For example: <script> window.addEventListener("load", () => { console.log("This function is executed once the page is fully loaded"); }); </script>
